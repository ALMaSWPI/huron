\doxysubsection{Detailed Description}
An array of FSR with values transmitted over Serial communication.

The sensor values are in double but sent in string in the following syntax\+: $<$sensor\+\_\+name$>$,$<$val\+\_\+1$>$,$<$val\+\_\+2$>$,...,$<$val\+\_\+n$>$~\newline
The sensor values should be sent periodically.
