This module contains ROS2 packages that support using ROS2 for simulation or hardware control. It includes the HURON description files (URDF/xacro), Gazebo 11 support, and example code. The ROS packages can be found in {\ttfamily src/}.

We are using \href{https://docs.ros.org/en/humble/index.html}{\texttt{ ROS2 Humble Hawksbill}}.

List of packages\+: 
\begin{DoxyItemize}
\item huron\+\_\+description\+: includes meshes, a macro xacro file to generate a HURON robot, a top-\/level xacro file to be visualized in RViz. 
\item huron\+\_\+gazebo\+: Gazebo 11-\/specific configuration, including sensor/control plugins and a Gazebo launch file. 
\item 

examples\+: Example of using HURON API with ROS2 {\bfseries{(TBA)}}


\end{DoxyItemize}
