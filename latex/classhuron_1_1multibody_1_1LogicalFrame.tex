\doxysubsection{Detailed Description}
A frame that is defined relative to another frame by an affine transformation. This transformation is user-\/defined using a function that takes the parent frame\textquotesingle{}s transform in world coordinates as an argument and returns the transform from the parent frame to this frame. 

\begin{DoxyNote}{Note}
This class can only be instantiated by the \mbox{\hyperlink{classhuron_1_1multibody_1_1Model}{Model}} class using Add\+Logical\+Frame().
\end{DoxyNote}
